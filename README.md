# campus
青春共享站是面向大学生的校园生活服务平台，以共享便捷为核心，助力学生高效获取信息，解决问题，构建便捷温暖的大学社区生态

#### 开发环境

* SpringBoot2
* Redis+Caffeine缓存
* Minio
* MySQL8
* vue3
* Uniapp
* SSE
* Sa-Token

#### 主要功能：

论坛帖子发布，校园活动，社团互动，交流学习，失物招领，AI对话等
